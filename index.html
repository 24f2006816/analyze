Data Processor & GitHub Pages Publisher

This project demonstrates an automated data processing pipeline using Python, Pandas, and GitHub Actions.

It takes an input data.xlsx file, converts it to data.csv, processes it with execute.py, and publishes the resulting result.json to GitHub Pages.

How it Works:

  • Data Preparation: data.xlsx is manually converted to data.csv.
  • Python Script: execute.py reads data.csv, performs data aggregation, and outputs result.json.
  • GitHub Actions CI/CD: A workflow automatically lints the code with Ruff, runs execute.py, and deploys result.json to GitHub Pages on every push to main.
